职位描述
诚聘嵌入式Linux系统工程师两名，提供具有竞争力的薪酬，欢迎广大有识之士加盟飞漫。


职责描述
- 负责公司产品在 Linux/VxWorks/ThreadX 等操作系统下的移植开发工作
- 负责 Linux 操作系统的定制开发
- 改进移植的流程和客户开发板的验证过程和方法
- 改进嵌入式平台的开发环境搭建的过程和方法
- 产品的安装和部署，应用程序的开发维护
- 编写并整理移植相关文档和产品相关编程开发指南

任职要求
- 熟悉嵌入式 Linux 操作系统，有独立完整移植嵌入式操作系统经验者优先。
- 了解嵌入式 Linux 内核定制、驱动程序开发。
- 熟悉嵌入式开发，擅长 C/C++，有三年以上嵌入式 Linux 开发经验。
- 熟悉一个或者多个平台上的 RootFS 和 GCC 编译工具链的制作。
- 熟练使用 Shell、Python、Perl 等脚本语言中的一种。
- 熟悉嵌入式系统中常用的图形库，熟悉 MiniGUI、Gtk、SDL 者优先。
- 熟悉嵌入式多媒体音视频库的移植定制，熟悉 GStreamer 者优先。
- 熟悉软件开发流程，有较强的团队合作精神。
- 大学本科以上学历，计算机相关专业。

　　北京飞漫软件技术有限公司（Beijing Feynman Software Technology Co., Ltd.，http://www.minigui.com）成立于 2002 年 9 月，由国际知名的自由软件项目 MiniGUI的创始人及主要开发者组建而成，是北京市高新技术企业，中关村创新技术企业，双软认证企业，致力于为嵌入式设备集成商或者制造商提供世界一流的嵌入式图形技术、嵌入式浏览器技术、嵌入式 JAVA 技术以及相关开发工具。

　　飞漫软件具有 10 年以上的嵌入式图形领域和5年以上的嵌入式浏览器的研发经验；丰富的嵌入式操作系统（嵌入式 Linux/uClinux、eCos、uC/OS-II、pSOS、ThreadX、VxWorks、Nucleus、OSE 等）和处理器架构（IA32、ARM、MIPS 等）的开发经验。

　　飞漫软件的主要产品有跨嵌入式操作系统和硬件平台的图形中间件 MiniGUI、所见即所得的可视化集成开发环境 mStudio、嵌入式浏览器 mDolphin，基于 Linux 内核的适用于中低端嵌入式设备的合璧操作系统 HybridOS 等。公司已经申请了多项产品发明专利，并取得了 28 项计算机软件著作权。

　　飞漫软件的产品历经了众多项目的验证，被国内外许多重量级客户选用，广泛用于手机、可视电话、PMP、机顶盒、智能家居、安防监控、医疗设备、电力、工控及仪表、军事工业等领域。用户遍及中国大陆、台湾地区、美国、意大利、韩国和日本等全球主要国家和地区，典型用户及合作伙伴有：华为、中兴通讯、大唐移动（联芯科技）、海信、飞思卡尔（美国）、长虹、TCL、联想、迈瑞、南瑞、炬力、Actinium（新加坡）、D2（美国） 等。值得一提的是，在中国自主开发的 3G 通讯标准 TD-SCDMA 中，约有 60% 获得入网许可证的 TD-SCDMA 手机使用 MiniGUI 作为其嵌入式图形平台，以支撑浏览器、可视电话等 3G 应用软件的运行。使用 MiniGUI 的主要 TD 手机品牌有中兴、联想、宇龙酷派等。

　　同时，飞漫软件陆续开放了 MiniGUI 1.6.10、Mesa3D on MiniGUI 等多款软件的源代码，已成为中国地区为开源社区贡献代码最多的软件企业，总体贡献代码行数已经超过 200 万行。

　　2006 年，飞漫软件被授予"2006 年度影响中国的嵌入式系统企业新星奖”，2007 年，魏永明总经理荣获“北京市第七届科技之光优秀创业企业家”美誉。面对种种荣誉，飞漫软件依然秉承“让嵌入式开发更容易”的服务理念，以严格的质量标准、积极的创新精神、专业的职业素质不断为客户提供完善的产品、培训以及技术支持，在满足用户的多样需求的同时帮助客户降低成本，加快产品二次开发速度，确保用户在激烈的市场竞争中取得领先地位，在分享客户的成功与荣誉的同时也体现出自己独特的价值。

　　2010 年，飞漫软件推出了基于 Linux 内核和飞漫专有软件的合璧操作系统（HybridOS）。依赖 HybridOS，飞漫软件将面向嵌入式开发企业，推出革命性的软件产品及服务。飞漫软件期待优秀人才的加盟。让我们共同努力，共创辉煌！
